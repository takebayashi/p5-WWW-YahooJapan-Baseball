# NAME

WWW::YahooJapan::Baseball - Fetches Yahoo Japan's baseball stats

# SYNOPSIS

    use WWW::YahooJapan::Baseball;

# DESCRIPTION

WWW::YahooJapan::Baseball provides a way to fetch Japanese baseball stats via Yahoo Japan's baseball service.
